**Convention for closing a Phase 1 item:**

1. Delete the bullet from `ROADMAP.md` (do **not** just flip the checkbox — open lines stay open).
2. Add a row to the table below with a fresh `P1-*` ID, a short summary, and the close date.
3. If the item touched shared specs (ARCHITECTURE.md, INTEGRATIONS.md, DESIGN_SYSTEM.md, etc.), reconcile the doc in the same commit so the spec and the code agree.
